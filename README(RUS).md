# 🤖 AI Бизнес Аналитик

**Генератор бизнес-требований (BRD) с AI** для ForteBank и корпоративных клиентов.

Проводит структурированные интервью, генерирует BRD с User Stories и Use Cases, оценивает качество и строит диаграммы в формате Mermaid с опциональным экспортом PNG.

## ✨ Функционал

### 🎯 Основное

* **Интерактивный диалог** – 10 шагов для сбора требований
* **Генерация BRD AI** – через GPT-4o
* **Поддержка типов инициатив**:

  * **Продуктовая** – опыт клиентов, метрики, монетизация
  * **Процессная** – оптимизация процессов, SLA, время цикла
  * **ИТ-система** – архитектура, интеграции, безопасность, надежность
  * **Соответствие и Риск** – регуляторные требования, контроль, смягчение рисков

### 📊 Структура BRD

Каждый BRD содержит:

* **Цель** – бизнес-цель
* **Описание** – контекст и описание проблемы
* **Объем** – включенный и исключенный объем
* **Бизнес-правила** – правила, ограничения, SLA
* **KPI** – ключевые показатели
* **User Stories** – требования в формате user story
* **Use Cases** – сценарии с основным и альтернативными потоками
* **Лидирующие индикаторы** – ранние индикаторы успеха

### ✅ Оценка качества

Автоматическая оценка:

* Полнота, ясность, согласованность, реализуемость, бизнес-ценность (0-100)
* Поиск недостающей информации
* Идентификация рисков
* Предложения дополнительных вопросов

### 📈 Визуализация

* **Процессные диаграммы** – блок-схемы
* **Диаграммы Use Case** – акторы и связи
* **Синтаксис Mermaid** – редактируемый
* **Экспорт PNG** – через Kroki (опционально)
* **Резерв** – исходный код Mermaid всегда доступен

### 📤 Экспорт и интеграция

* Markdown
* HTML с встроенными диаграммами
* Создание страниц в Confluence
* Скачиваемый код Mermaid

## 🚀 Установка

### Требования

* Python 3.10+
* pip
* OpenAI API Key
* Опционально: токен Confluence

### Установка

```bash
git clone <repo-url>
cd AI_BusinessAnalyst
python -m venv venv
# Windows PowerShell: venv\Scripts\Activate.ps1
# Linux/macOS: source venv/bin/activate
pip install -r requirements.txt
```

Проверка:

```bash
python -c "import streamlit; import openai; print('✅ OK')"
```

## ⚙️ Настройка

### OpenAI API Key

> Ключ задается **в коде**:

```python
import openai
openai.api_key = "sk-your-key"
```

### Confluence (опционально)

Поля боковой панели:

| Поле           | Описание                 |
| -------------- | ------------------------ |
| URL            | Базовый URL              |
| Email          | Email Atlassian аккаунта |
| API Token      | Токен                    |
| Space Key      | Пространство             |
| Parent Page ID | Родительская страница    |

### Генерация PNG (опционально)

Включите "Генерировать PNG через Kroki" в боковой панели.

## 📖 Использование


   * Диаграммы Mermaid
   * Опциональные PNG через Kroki
